cen::library::library ( )
inline

Initializes the library.

Note
Make sure to have the library instance as a local variable that will outlive the duration of your main program. It's not sufficient to just call the constructor but not store the result as a variable.
Precondition
there mustn't exist any other instances of this class at the time of invocation of this constructor.
Exceptions
sdl_errorif the core SDL2 library can't be initialized.
img_errorif the SDL2_image library can't be initialized.
ttf_errorif the SDL2_ttf library can't be initialized.
