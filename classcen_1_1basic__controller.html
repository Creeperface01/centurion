◆ basic_controller() [3/3]

template<typename BB = B, detail::is_owner< BB > = true>
cen::basic_controller::basic_controller ( const int  index = 0)
inlineexplicit

Attempts to create a game controller.

The joystick index is the same as the device index passed to the joystick constructor. The index passed as an argument refers to the n'th game controller on the system.

Note
The supplied index is not the value which will identify the controller in controller events. Instead, the joystick's instance id (SDL_JoystickID) will be used.
Remarks
This constructor is only available for owning game controllers.
Parameters
indexthe device index, can't be >= than the amount of number of joysticks.
Exceptions
sdl_errorif the game controller cannot be opened.
Since
5.0.0

static auto cen::basic_controller::load_mappings ( const not_null< czstring >  file) -> std::optional<int>
inlinestaticnoexcept

Loads a set of game controller mappings from a file.

A collection of game controller mappings can be found at here. New mappings for previously known GUIDs will overwrite the previous mappings. Furthermore, mappings for different platforms than the current platform will be ignored.

Remarks
It's possible to call this function several times to use multiple mapping files.
Note
The text database is stored entirely in memory during processing.
Parameters
filethe path of the mapping file.
Returns
the amount of mappings added; std::nullopt if something went wrong.
Since
5.0.0

auto cen::basic_controller::rumble ( const u16  lo,
const u16  hi,
const milliseconds< u32 >  duration 
) -> bool
inline

Starts a rumble effect.

Calls to this function cancels any previously active rumble effect. Furthermore, supplying 0 as intensities will stop the rumble effect.

Note
This function has no effect if rumbling isn't supported by the controller.
Parameters
lothe intensity of the low frequency motor.
hithe intensity of the high frequency motor.
durationthe duration of the rumble effect.
Returns
true on success; false otherwise.
