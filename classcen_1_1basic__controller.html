Detailed Description

Represents a game controller, e.g. Xbox or Playstation controllers.

This class is a part of the owner/handle API, the owner/handle versions of this class are controller/controller_handle

You may need to load appropriate game controller mappings before you can begin using the game controller API with certain controllers. This can be accomplished using the cen::hint::controller::config_file hint, or the load_mappings() and add_mapping() functions.

For a community managed database file of game controller mappings, see https://github.com/gabomdq/SDL_GameControllerDB (if the link doesn’t work for some reason, you should be able to find a copy in the Centurion test resources folder).

◆ basic_controller() [3/3]

template<typename TT = T, detail::is_owner< TT > = 0>
cen::basic_controller::basic_controller ( const int  index = 0)
inlineexplicit

Attempts to create a game controller.

The joystick index is the same as the device index passed to the joystick constructor. The index passed as an argument refers to the n'th game controller on the system.

Note
The supplied index is not the value which will identify the controller in controller events. Instead, the joystick's instance id (SDL_JoystickID) will be used.
Remarks
This constructor is only available for owning game controllers.
Parameters
indexthe device index, can't be >= than the amount of number of joysticks.
Exceptions
sdl_errorif the game controller cannot be opened.
