Detailed Description

Represents a two-dimensional point.

This class is designed as a wrapper for SDL_Point and SDL_FPoint. The representation is specified by the type parameter.

Note
This point class will only use int or float as the actual internal representation.
Template Parameters
Tthe representation type. Must be convertible to int or float.
