Detailed Description

template<typename Hint, typename UserData = void>
class cen::hint_callback< Hint, UserData >

Represents a handle for dealing with hint callbacks.

This class provides methods for easily connecting and disconnecting a callback from receiving updates.

Template Parameters
Hintthe type of the associated hint. Should be one of the hint types defined in the hints.hpp header.
UserDatathe type of the user data, defaults to void.
