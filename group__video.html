◆ blend()

auto cen::blend ( const color &  a,
const color &  b,
const double  bias = 0.5 
) -> color
inline

Blends two colors according to the specified bias.

Precondition
bias should be in the range [0, 1].

This function applies a linear interpolation for each color component to obtain the blended color. The bias parameter is the "alpha" for the interpolation, which determines how the input colors are blended. For example, a bias of 0 or 1 will simply result in the first or second color being returned, respectively. Subsequently, a bias of 0.5 will blend the two colors evenly.

Parameters
athe first color.
bthe second color.
biasthe bias that determines how the colors are blended, in the range [0, 1].
Returns
a color obtained by blending the two supplied colors.
Since
6.0.0
