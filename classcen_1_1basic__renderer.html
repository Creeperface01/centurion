◆ draw_lines()

template<typename Container >
void cen::basic_renderer::draw_lines ( const Container &  container)
inlinenoexcept

Renders a collection of lines.

This function requires the the Container type provides the public member value_type and subsequently, that the value_type in turn provides a value_type member. The former would correspond to the actual point type, and the latter corresponds to either int or float.

Warning
Container must be a collection that stores its data contiguously! The behaviour of this function is undefined if this condition isn't met.
Template Parameters
Containerthe container type. Must store its elements contiguously, such as std::vector or std::array.
Parameters
containerthe container that holds the points that will be used to render the line.
