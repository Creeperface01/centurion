Detailed Description

Represents a sound effect.

The supported file formats are the following:

  • WAVE/RIFF (.wav)
  • AIFF (.aiff)
  • VOC (.voc)
  • OGG (.ogg)
  • VOC (.voc)
Usage
Usage of this class is pretty straightforward and self-explanatory. The fundamental methods are play() and stop(), with additional support for effects such as fading and looping.
